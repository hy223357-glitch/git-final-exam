Toi dang hoc ve Git và GitHub 
Toi yeu thich viec tim hieu cach lam viec nhom bang Git.
Muc tieu cua toi la tro thanh mot lap trinh vien gioi va chuyen nghiep.

So thich:
- Lap trinh va nghien cuu cong nghe moi.
- Choi game va xem phim.
- Du lich kham pha.

Ky nang:
- Lap trinh web (HTML, CSS, JS).
- Su dung Git và GitHub.
- Giao tiep va lam viec nhom.

Kế hoạch học tập:
- Ôn lại toàn bộ kiến thức Git và GitHub.
- Thực hành các lệnh Git cơ bản (add, commit, push, pull).
- Học cách tạo, review và merge pull request.
- Nghiên cứu thêm về branching và workflow trong Git.
- Ứng dụng Git để làm việc nhóm trong đồ án cuối kỳ.
